# SentinelDLP v1.1.0

**Release Date:** Incremental Update
**Status:** Stable

## Overview

Enhanced UI/UX release with improved visualization and user experience features.

## New Features

### Dashboard Enhancements
- Interactive sensitivity gauge with color-coded levels
- Expandable findings sections
- Improved results layout

### File Handling Improvements
- Enhanced drag-and-drop feedback
- File type validation on frontend
- Improved error messaging

### UI Components
- Custom icon library (Lucide-style)
- Responsive design improvements
- Loading states and animations

## Changes from v1.0.0

### Added
- Visual sensitivity gauge component
- Department relevance chips
- Regulatory concerns display
- Recommended actions section

### Improved
- Error handling and user feedback
- Mobile responsiveness
- Overall UI polish

## Upgrade Notes

No breaking changes from v1.0.0. Direct upgrade supported.

```bash
docker compose pull
docker compose up -d
```
